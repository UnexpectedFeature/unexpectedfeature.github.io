# Hexo Quick Guide

## Quick Commands

### Create a new post

``` bash
# Command
hexo new [layout] <title>
# Example
hexo new post "title"
```

More info: [Writing](https://hexo.io/docs/writing.html)

### Run server

``` bash
hexo server
```

More info: [Server](https://hexo.io/docs/server.html)
